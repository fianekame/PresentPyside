Membuat Revisian Pertama

Dengan


							git add file1.html

							git add file3.html

Atau


							git add file1.html file2.html file3.html
						

							git add *.html
							git add .

							git commit -m "Comitan Pertamaku"

Melihat Diff Revisian

Beberapa Fungsi Log Revisian

 

								git diff

								git diff nomercommit nomercommit

Membatalkan Revisian

  1. Modified File ? git checkout
  2. Stagged File ? git reset
  3. Commited File ? git revert

Git Checkout


								git checkout namafile
Kembali ke yang lalu ? 

								git checkout nomercommit

Apa Yang Dilakukan ?

Git Reset

file yang sudah di add (git add), namun belum disimpan (git commit) ke dalam Git. 

								git reset namafile
Kembali ke yang lalu ? 

								git reset --type nomercommit
  1. Modified File ? --mixed
  2. Stagged File ? --soft
  3. Commited File ? --hard

Apa Yang Dilakukan ?

Git Revert


								git revert -n nomercommit
Revert artinya mengembalikan. Revert akan akan mengambil kondisi file yang ada di masa lalu, kemudian menggabungkannya dengan commit terakhir.

Apa Yang Dilakukan ?
